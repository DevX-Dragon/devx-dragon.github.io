## What it does

Fake terminal. Real code. You can type commands and it responds. Tab completion works. Arrow-up for history works. I'm unreasonably proud of this.

## Built with

Zero dependencies. Raw DOM manipulation. CSS animations for the scanline and cursor blink. It taught me more about event handling than any tutorial ever did.
